# ProsperLoan EDA Project
# ProsperLoan探索性数据分析

## 项目介绍 

Prosper是美国的一家P2P消费贷款公司，项目使用R语言，通过对prosperLoan2005-2014年的贷款数据进行探索发现数据间的联系。
## 具体分析过程  

1.从原始数据prosperLoanData.csv中提取相关变量组成数据集。  
2.选择数据进行统计分析和单变量绘图分析。  
3.选择数据进行双变量绘图分析，主要探索各数据与LoanOriginalAmount之间的关系。  
4.选择数据进行多变量绘图分析和回归分析。
5.定稿和反思。  
项目代码保存在prosperloan_EDA.RMD中，分析及绘图保存在prosperloan_EDA.html中。
